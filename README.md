# 个人网站原型

这是一个静态个人网站，不需要数据库、不需要后台，也不需要安装复杂工具。

## 怎么查看

直接双击 `index.html`，或在浏览器中打开这个文件。

## 怎么改成你的内容

- 网站标题：修改 `index.html` 里的 `<title>` 和首页大标题 `你的名字`
- 个人介绍：修改 `About` 区域
- 文章卡片：修改 `Thoughts` 区域的 `<article>`
- 笔记列表：修改 `Notes` 区域的 `<li>`
- 图片：替换 `Frames` 区域的图片链接
- 联系方式：修改 `Hello` 区域的邮箱和链接

## 网址能不能自定义

可以。

- 免费网址：可以用 GitHub Pages，例如 `https://你的用户名.github.io`
- 自定义网址：可以买一个域名，例如 `yourname.com`，然后绑定到 GitHub Pages 或 Vercel

第一版建议先用免费网址上线，等内容稳定后再买自定义域名。
